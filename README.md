# App ArzSuite

Proyecto en Flutter (Material 3) soportado por Riverpod y Supabase, diseñado para conectar con el backend "centro-backend" en CakePHP 5.

## Estructura de la documentación

La lógica de negocio, estructuras, y detalles de arquitecturas pueden consultarse en los siguientes archivos (ubicados en la carpeta `git/docs/`):

- **[Lógica de Negocios](git/docs/logica_de_negocios.md)**: Cómo funciona internamente cada módulo de la parte visual conectada a nuestros micro-servicios.
- **[Estructura de Carpetas](git/docs/estructura_carpetas.md)**: Una explicación para la lectura rápida en dónde modificar estilos y la arquitectura Riverpod implementada.
- **[Conexiones](git/docs/conexiones.md)**: El set de credenciales de API para interactuar con Supabase e información sobre centro-backend.
- **[Tech Stack](git/docs/tech_stack.md)**: La deconstrucción rápida de todo el sistema.

*Este README se irá actualizando de forma dinámica conforme la app vaya creciendo.*
