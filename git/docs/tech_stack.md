# Tech Stack (Pilas Tecnológicas)

El stack con el que se construye esta aplicación es el siguiente:

1. **Framework Frontend:** Flutter (Versión ^3.0+)
2. **Diseño Visual:** Material 3 (Totalmente implementado desde `app_theme.dart`).
3. **Gestión de Estado e Inyección de Dependencias:** Riverpod (`flutter_riverpod`).
4. **Backend/BaaS:** Supabase (`supabase_flutter`).
5. **Consumo de APIs Externas:** Dio (Para conectar fluido con CakePHP 5 - `centro-backend`).
6. **Lógica Externa (Backend Ecosistema):** CakePHP 5 (Sistema Aislado: *centro-backend*).
